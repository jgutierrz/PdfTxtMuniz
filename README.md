# Pdf_txt_Muñiz

Este proyecto sigue una arquitectura estándar de desarrollo en Python.
El entorno virtual está pre-configurado para VS Code.

## 📂 Guía de Carpetas

| Carpeta | Descripción y Uso |
| :--- | :--- |
| **`src/`** | **Source Code.** Lógica del programa. |
| **`data/raw/`** | **Datos Crudos.** Archivos originales. NO modificar. |
| **`data/processed/`** | **Datos Procesados.** Resultado final. |
| **`.vscode/`** | **Configuración.** Reglas automáticas para el editor. |

## 🚀 Instalación Rápida
1. `python -m venv venv`
2. Reinicia VS Code o abre una nueva terminal.
3. Verás `(venv)` activo automáticamente.
4. `pip install -r requirements.txt`
